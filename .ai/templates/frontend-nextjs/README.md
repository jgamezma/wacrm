# Frontend — Next.js template

Next.js (App Router) + TypeScript frontend. Extends the
[base project template](../project/README.md).

## What it adds on top of `project`

- `.ai/stack-frontend-nextjs.md` — tooling, layout, and Next.js conventions.
- `.cursor/rules/frontend.mdc` — Cursor rules scoped to TS/TSX files.

## Use it

```bash
scripts/sync-template.sh init frontend-nextjs /path/to/my-web
```

## Update later

```bash
scripts/sync-template.sh update /path/to/my-web
```

See the [top-level README](../../README.md) for the full model.
